Recomendações para o cliente

Após preencher os dados, o cliente deve estar ciente dos cuidados para que a instalação ocorra sem atrasos.

Antes da instalação

  • Mesa reservada para o equipamento.
  • Tomada 110V próxima.
  • TV própria com HDMI funcionando, se aplicável.
  • Responsável presente para receber as orientações.

Durante o evento

  • Não alterar configurações.
  • Evitar líquidos próximos aos equipamentos.
  • Não bater, soprar ou gritar nos microfones.
  • Supervisionar crianças durante o uso.
✅ Ao finalizar o cadastro, o cliente declara ciência das recomendações acima.
